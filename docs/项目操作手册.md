# 数据分析 Agent 项目操作手册

**版本**: 2.0.0  
**同步日期**: 2026-04-02

## 1. 快速开始

```bash
cd /workspaces/data
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

## 2. 启动方式

### 2.1 LangGraph Studio（推荐）

```bash
langgraph dev
```

- API: `http://localhost:2024`
- Studio: `http://localhost:2024`

### 2.2 REST API

```bash
python scripts/run_api.py
```

- Swagger: `http://localhost:8000/docs`

## 3. 核心接口

### 3.1 对话

`POST /api/chat`

请求示例：

```json
{
  "message": "统计最近7天告警数量",
  "thread_id": "session_001",
  "stream": false,
  "role": "analyst"
}
```

### 3.2 流式对话

`POST /api/chat/stream`

### 3.3 会话清理

`DELETE /api/chat/{thread_id}`

### 3.4 HITL 恢复

`POST /api/resume`

请求示例：

```json
{
  "thread_id": "session_001",
  "decision": "edit",
  "edited_sql": "SELECT * FROM device LIMIT 100"
}
```

支持决策：
- `approve`
- `edit`
- `reject`

### 3.5 健康与工具

- `GET /api/health`
- `GET /api/ready`
- `GET /api/live`
- `GET /api/tools`
- `GET /api/tools/{category}`

## 4. 生产配置建议

- 开启日志：`LOG_LEVEL=INFO` 或 `DEBUG`
- 敏感环境开启 HITL：`ENABLE_HITL=true`
- 需要中断恢复时使用 sqlite checkpointer：

```env
CHECKPOINTER_BACKEND=sqlite
CHECKPOINTER_PATH=./.langgraph_checkpoints.sqlite
```

说明：sqlite 依赖不可用时会自动回退到内存 checkpointer。

## 5. 常见问题

1. `resume failed`：先确认该 `thread_id` 之前确实触发过 HITL 中断。
2. SQL 被拒绝执行：检查是否包含写操作关键字。
3. 无法命中正确表：可先调用 `rebuild_data_semantic_layer` 刷新语义层。
4. 启动失败：优先检查 `.env` 中模型与数据库配置。
